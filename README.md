# 💼 Portfólio Web – Vinicius Fernandes

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um **portfólio pessoal**, apresentando minhas habilidades técnicas, projetos pessoais e informações de contato de forma profissional na web.

O site foi desenvolvido utilizando **tecnologias fundamentais de desenvolvimento web**, com foco em organização de código, boas práticas de estruturação e experiência do usuário.

---

## 🎯 Objetivos do Projeto

- Praticar **desenvolvimento web front-end**
- Apresentar minhas **habilidades técnicas**
- Disponibilizar links para **GitHub e LinkedIn**
- Exibir informações profissionais

---

Este projeto foi desenvolvido utilizando as seguintes tecnologias:
